| Versão | Suportado          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Relatando uma Vulnerabilidade

Levamos a segurança do Offstream a sério. Se você descobrir uma vulnerabilidade de segurança, informe-nos imediatamente.

**NÃO** crie um problema público no GitHub para vulnerabilidades de segurança.

Em vez disso, envie um e-mail para [security@example.com] (substitua pelo e-mail real, se disponível, ou instrua a abrir um comunicado privado).

Verificaremos a vulnerabilidade e trabalharemos em uma correção. Agradecemos sua cooperação na divulgação responsável de vulnerabilidades.
